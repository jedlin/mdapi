Jobs API

Available endpoints:

  • [get] /api/jobs - returns all jobs
  • [post] /api/jobs - add a job to the database
  • [get] /api/jobs/:job_id - return data for job identified by job_id
  • [put] /api/jobs/:job_id - update data for job identified by job_id
  • [delete] /api/jobs/:job_id - delete job identified by job_id
